準拠：`世界設計・全体図v1.md`（§5-1 晴れの海=光=セレネバッジ）、`ヴォイス編・全体設計v2.md`（§2 山1＝ジム4後にキヨハラ戦①）。

## 2. ジムのしかけ：光の回廊（★方針・実装可）
「鏡（プリズム）に光を通すと、暗い床が“光の橋”になって渡れる」パズル。
- 実装はジム3の**セグメント方式（バルブXOR）を流用**：`prism_1` / `prism_2` の2つのプリズムスイッチ（pickupsフラグ）で、床セグメントが **暗闇(通行不可) ⇄ 光の橋(通行可)** を切り替える。
- ロジック例（☆）：橋Aはprism1、橋Bはprism2、中央の宝への橋CはXOR（片方だけON）。両方ONでリーダーへ到達。
- 反則防止：ジム3同様にBFSで各状態の到達性・宝の隔離・行き止まり健全性を検証。
- タイル（新規）：`120 ジム床(星石)` / `121 光の橋(点灯・歩行可)` / `122 暗闇(通行不可)` / `123 プリズム装置`。BootSceneに描画分岐を追加。

## 3. 光の新アルモン：ルナルクス（🎨アート要）
- **ホタルナ → ルナルクス**（Lv30台で進化。図鑑計画v1に既定）。
- コンセプト：ホタルナ（蛍のような光の子）が成長した、月光をまとう気高い姿。三日月と星のモチーフ。
- 役割：光のエースアタッカー。とくこう高め・すばやさ高め。
- 🎨アート指示（前作モンスターと同じ緑背景でOK）：
  - 正面（バトル敵・手持ち・図鑑用）＋背面（自分の繰り出し用）
  - サイズ感は既存のプロミネンス級のエース。淡い青白〜金の発光、三日月の角/尾。
- 実装：`monsters.json` に追加（No.029）、`encounters` は不要（進化限定 or 終盤レア）、`ホタルナ`の evolves に設定。

## 4. キヨハラ戦①（★構造確定・キヨハラ絵は実装済み）
- 位置：**ジム4クリア直後**、セレネタウンで発生（ヴォイス編v2 山1）。
- 形式：**イーゼン共闘のダブルバトル**（味方＝主人公＋イーゼン想定の演出／実装は既存doublesを流用し、敵側キヨハラ＋したっぱ or キヨハラ2体）。
  - 簡易案（☆）：`doubles: true` のトレーナー1エントリ「キヨハラ＆したっぱ」。イーゼンの共闘は前口上の演出（セリフ）で表現し、戦闘自体は既存プレイヤー2体先頭で対応。
- キヨハラ型：**金属/氷**（v2§1）。Lv38〜42・エース金属（v2§4）。
