## 1. 位置づけ・接続 ★決定ベース

ワールド接続図（`世界設計・全体図v1.md` §5-2）より:

```
クレーターシティ ★ジム1(砂/リューマ) ──砂場ルート2──▶ ネクタルタウン ★ジム2(氷)
                                                          │
                                                          ▼（将来）次の海へ
```

- **町の正式名**：ネクタルタウン（英名 Nectaris / 神酒の海）
- **南の入口**：砂場ルート2の**右上の未開放出口 (20,0)/(21,0)** をここへ接続する
  （現状 `MapScene.SEALED_EXITS` で封鎖中 → 実装時に warp を追加して開通）
- **（将来）別の出口**：北 or 東に、次エリア（豊かの海タウン ★ジム3(炎) へ向かうルート/洞窟）への口を1つ確保しておく（今は未開放でよい）

### 接続実装メモ（実装時）

  `{x:20,y:0 / x:21,y:0 → targetMap:"nectar_town", targetX/Y:<南入口>}` を追加。

## 2. コンセプト・教育ネタ ☆提案（要 最終裏取り）

- **舞台**：神酒の海（Mare Nectaris）。月の**近い側 南部**、静かの海の南西に位置する小さめの海。
- **教育フック**：
  - 地質時代 **「神酒代(Nectarian)」の基準**になった由緒ある盆地（約39億年前級）。
  - 盆地の縁に走る断崖 **アルタイの崖（Rupes Altai）** が有名。
  - 南縁の **フラカストリウス（Fracastorius）** クレーターは片側が溶岩で埋まった「割れた指輪」状。
- **看板/NPCで伝える1行例**：「ここは 神酒の海。月の 大むかしを 知る 手がかりが ねむる 場所だ。」
- ※本書の月の事実は一次調査ベース。実装前に JAXA/NASA 等で最終確認（世界設計書の方針に準拠）。

## 3. 見た目・テーマ ☆提案

- ジム2が **氷タイプ** のため、町全体を**霜／低温の探査前哨基地**の質感に。
  - レゴリス地面＋**霜（うっすら白い）タイル**、氷結したパイプ／冷却塔（点景）、青白い照明。
  - 「神酒の海は寒暖差が激しく、影の部分は極低温」→ 氷ジムの理由づけ（設定）。
- 既存の砂タイル・岩に加え、**霜タイル**を1〜2種 追加（`tileTypes` に walkable な氷床風タイルを新設）。
- 縮尺は既存踏襲（建物 footprint 方式、2.5倍）。

## 4. マップ規模・ゾーン構成 ☆提案

- **サイズ**：24×22 前後（ホームのクレセントタウン=20×18 より少し大きく、クレーターシティ=36×36 より小さい「町」規模）。
- **レイアウト方針**：クレーターシティ再設計と同じく、四角い壁ではなく地形で不定形の輪郭。曲がった道でゾーンをつなぐ。

```
              [北口→（将来）次の海ルート]
   ┌──────────────────────────────┐
   │  ★ジム2(氷)         : 町の奥・一段高い区画        │
   │  リカバリーポッド    : 入口近く（全滅復帰先の候補）    │
   │  プラネットショップ  : 商業（氷/寒冷装備の品ぞろえ）    │
   │  住居 ×2〜3          : うち1件は会話イベント/ヒント     │
   │  教育モニュメント    : アルタイの崖 の展望/看板        │
   └──────────────────────────────┘
              [南口→砂場ルート2]
```

- **入口（南）**：ルート2から入る。すぐ近くにリカバリーポッド（旅の再出発点）。
- **奥（北）**：ジム2。手前にジムゲート（ジム内トレーナーを倒す等の条件は？未決）。
