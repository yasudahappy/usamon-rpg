# うさもんRPG 仕様書 v2

リポジトリ: https://github.com/yasudahappy/usamon-rpg

※ v1からの主な追加:進化系統、5体の技表(Lv1〜Lv100対応)、進化に合わせた技レベルの調整、進化する/しないを選べる仕組み。

---

## 1. コンセプト

ポケモン ルビー・サファイア風の2D見下ろし型RPG。
舞台は**月面**。プレイヤーはモンスター「うさもん」を相棒に、月面の砂場で野生モンスターを捕まえ、育てて戦い、最終的に**四天王を倒して殿堂入り→火星への道が開く**。

- まず**ミニ版(MVP)**を完成させ、その後**完成版**へ拡張する
- 拡張を前提としたデータ設計にする

---

## 2. 世界観・舞台

- **月面**が舞台。基地、街、砂場(エンカウントエリア)、クレーター、ダンジョン
- 草むらの代わりに**「砂場(レゴリス地帯)」**
- 殿堂入りすると別の惑星(火星)に行ける世界観

---

## 3. 主人公

- 性別(男性/女性)選択
- 宇宙服5色(ホワイト/ブルー/オレンジ/ピンク/ブラック)
- 歩行グラフィック:上下左右4方向 × 歩きアニメ

---

## 4. タイプ相性(8タイプ・4つずつ2サイクル)

### サイクルA(自然系)
**炎 → 氷 → ガス → 砂 → 炎**

### サイクルB(エネルギー系)
**光 → 影 → 電 → 金属 → 光**

### ダメージ倍率
- 有利: 2.0倍 / 不利: 0.5倍 / それ以外: 1.0倍
- サイクルAとBの間は常に等倍

---

## 5. レベル・成長システム

- 最大レベル:Lv100
- レベルアップで HP/こうげき/ぼうぎょ/すばやさ が上昇
- 新しい技を覚える(最大4つ、入れ替え選択)
- 一定レベルで進化(キャンセル可能)

## 9〜12. システム・技術

- 移動:見下ろし2D、キーボード+スマホタッチ両対応
- バトル:ターン制コマンド、タイプ相性、捕獲システム
- 手持ち最大6体、セーブ:localStorage
- 技術:Next.js + TypeScript + Phaser 3、GitHub Pages
- データはJSONで管理（拡張容易）
