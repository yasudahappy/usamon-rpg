## 2. ストーリーの背骨 ★決定ベース
- 主人公（23歳・性別/宇宙服選択）は**月面開発プロジェクト**に選ばれて月へ。
- **キノシタ博士**から売れ残りの**うさもん(光)**を譲り受け、月面を冒険。

## 4. タイプ（8種・2サイクル）＝ジム設計の基盤 ★決定
- **サイクルA（自然系）**：炎 → 氷 → ガス → 砂 →（炎へ）　各々「次」に2.0倍
- **サイクルB（エネルギー系）**：光 → 影 → 電 → 金属 →（光へ）　各々「次」に2.0倍
- **8タイプ = 8ジム**にちょうど対応させられる（本書の肝）。
- スターターは**うさもん(光)**。※光は金属に弱い(金属→光=2.0)ので、**金属ジムは後半**に置くのが安全。

## 6. 現行実装との対応・リネーム方針（？要決定）
現状のマップ名は「本物の月の地名」ではなく仮名の可能性がある。以下いずれかを選択：

- **案A（推奨）**：現行3マップを**静かの海 地方**として位置づけ、名前は据え置き。
  - クレセントタウン＝静かの海のホーム集落／砂場ルート1＝そのまま／クレーターシティ＝静かの海地方のジム1タウン

- **案B**：地名を実在名に寄せて改名（例：クレーターシティ→静かの海シティ 等）。整合は綺麗だが実装・セーブ・参照の修正コスト大

> ムーンベース(moonbase)＝居住区画は「静かの海の開発基地」として残す。§5-2はこの案Aを前提に描いている。

## 7. クレーターシティ 再設計スペック（☆提案・次の実装対象）
**狙い**：現状の「四角い壁＋大広場に建物押し込み」を、参考写真のような**有機的で複雑な街**にする。

### 7-1. 方針
- **外周を壁→地形に**：レゴリス平原(60)・岩・小クレーター・段差で**不定形の輪郭**にする（今の四角い柵の全周は廃止）。柵は**区画の縁取り（農園・広場まわり）**にだけ残す。
- **大広場→曲がりくねった道＋ゾーン分け**。道(40)で各ゾーンをつなぎ、間をレゴリス/岩で埋める。
- **マップ拡張**：28×28 →**36×36前後**（余白と道を作る）。
- **出入口を複数**：南＝砂場ルート1、（将来）東 or 北＝ルート2（次の海へ）。

### 7-2. ゾーン構成（☆提案）
```
                 [北口→ルート2(将来)]
   ┌───────────────────────────────┐
   │  シビック区   :  ★ジム / メディカルセンター       │
   │  中央広場     :  ランドマーク(アポロ着陸船モニュメント)  │
   │  商業区       :  プラネットショップ / 資材倉庫        │
   │  住宅区       :  住居×数件(入室可。うち1件=配達先)      │
   │  インフラ区   :  発電所 / 通信ステーション(点景)       │
   │  農園ドーム   :  食料培養室(緑の区画・柵で縁取り)       │
   │  隕石サイト   :  町外れ。ジム1攻略後に隕石→洞窟入口が出現 │
   └───────────────────────────────┘
                 [南口→砂場ルート1]
```
- ランドマーク＝**アポロ11号 着陸船のモニュメント**（教育＆「静かの海」らしさ、街の象徴）。
- 既存建物（ジム/メディカル/ショップ/リカバリーポッド/住居×2）は**再配置**して各ゾーンへ。**住居は数件に増やす**（街らしさ＋配達先を1件確保）。
- **隕石サイト（町外れ）**を確保：ジム1攻略後のイベントで隕石が落ち、**洞窟入口が出現**（`ストーリー脚本v1.md`第4章）。この洞窟＝炎タイプ初登場ダンジョン。
- 施設の追加は「1つずつ・全部使わなくてOK」の方針を継続。
- **研究所（ムーンベース）に研究員NPCを2人追加**（会話用）。

### 7-3. 実装メモ（技術）
- マップJSONの floor/collision を作り直し。地形タイル：レゴリス(60)・舗装/道(40/61)・岩(6)・クレーター(7)・農園の芝(62)を活用。
- 建物は今の footprint 方式（透過PNGを footprint に合わせて描画、下地は地面タイル）を踏襲。
- 縮尺は統一済みの 2.5倍を維持。小さすぎない配置に。
