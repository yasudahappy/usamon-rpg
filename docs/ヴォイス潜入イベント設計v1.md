# ヴォイス潜入イベント 設計v1（ネクタルタウン）

## 0. 位置づけ
- **舞台**：ネクタルタウン（既存マップに追い込みイベントを重ねる／新マップは作らない）
- **接続**：コジマの警告「南極の永久影の水を狙う 不審な者たち」＋ `gym_2` の立ち聞きカットシーン（フラグ `nectar_voice_seen`）の回収編。
- **タイミング**：ネクタルバッジ入手後（ジム2クリア）＝フラグ `nectar_gym_cleared` を新設し、これを起点に発火。
- **コンセプト**：教育を止めない。ヴォイスは「月の水資源を独占して売る」現実の“宇宙資源・水の利用”テーマを、悪役の口から子ども向けに提示する装置。

## 2. イベントの流れ（潜入 → 制圧）
1. **発火**：ジム2クリア後、ネクタルタウンに戻る／ジムを出ると短いカットシーン。
   - 街の照明が緑色に明滅（ヴォイスカラー）。住人がざわつく。
   - キヨハラ（参謀）が広場中央でアナウンス：「この街の水は われらが 管理する。逆らう者は 実力で 排除する。」
2. **街のジャック**：4人の団員が街の要所を封鎖（下表）。プレイヤーは1人ずつ倒して道を開ける。
   - 各団員は「なぜ水が大事か／月の水はどこにあるか」を歪めて語る → 倒すと正しい豆知識で上書き（教育の“反転”演出）。
3. **参謀キヨハラ戦**：4人を倒すと広場が開放。キヨハラと対決。
4. **団長シノブ登場**：キヨハラ撃破後、シノブが現れ捨て台詞＋次エリア（豊かの海／南極方面）への引き。実戦は次章に持ち越し（イラスト到着後に確定）。
5. **収束**：街の照明が戻り、住人がお礼。研究者から次の目的地のヒント。

## 4. 新規フラグ / セーブ
- `nectar_gym_cleared`：ジム2クリア時に立てる（発火条件）。
- `voice_raid_started` / `voice_grunt_a..d_defeated` / `voice_kiyohara_defeated` / `voice_raid_cleared`：進行管理。
- 撃破済み団員は再戦なし（`defeatedTrainers` に準拠）。

## 5. 教育フック（団員セリフの“反転”設計・下書き）
- 団員A：「水があれば ロケットの 燃料も 作れる。だから 独り占めするのさ。」→撃破後の住人/研究者「水は 水素と 酸素に 分けると 燃料に なる。でも “みんなの もの”だよ。」
- 団員B：「月の水は 日の当たらない クレーターの 底に 凍って ねむってる。」→ここは事実。正しい知識を悪役が言い、"使い方"が悪いという構図。
- 団員C：「太陽が 当たらないから マイナス170ど。だから 何億年も 溶けずに 残る。」→事実＋「だから貴重＝独占したい」動機。
- 団員D：「水は 飲み水にも 農業にも 使える。月に 住むなら 一番の 宝だ。」→撃破後「宝だからこそ 分け合うんだ。」
