# 宇宙資源ビジネス編 v1

くものうみタウン（雲の海）に、月資源をあつかう会社「月資源開発」の
社長 **タカ** と 研究責任者 **ミホ** が登場する。実在の月資源を
子ども向けに紹介しつつ、ヴォイス団の「水のひとりじめ」という
既存の対立軸に「みんなのための月」というテーマでつなげる。

## 教育要素（実在の月資源）

- **ヘリウム3**：太陽風が長い年月をかけて月のレゴリスに蓄えた希少ガス。
  将来のクリーンエネルギー（核融合燃料）候補。
- **レゴリス（月の砂）**：鉄・チタン・アルミなどの金属や酸素を含み、
  建材・ロケット材料・3Dプリント資材になりうる。
- **水氷**：南極の永久影クレーターに氷が残る。水を分解すると
  水素＋酸素になり、ロケット燃料・呼吸用の空気になる。月で生きる鍵。

## 実装

- `MapScene.placeCloudResourceBiz()`：cloud_town の create ブロックから呼び出し。
  タカ・ミホ・資源サンプル展示台（`cloud-resource-display` テクスチャ）を配置。
- タカ/ミホは `nectarExam`（＝当該タイルが固定＋話しかけ可能）で配置。
- 展示台カード：「月の たからは、あらそう ためでなく、みんなで つかうために。」
