## A. ストーリー・世界観(物語の骨格)

### ★決定した大方針:「本物の月」を舞台にする教育×冒険ゲーム
- **地名・地図は実際の月面の正しい情報を使う**（静かの海、コペルニクスクレーター、月の裏側、南極シャクルトンクレーター など。実在の地名・地形は事実なので自由に使える）
- **ストーリーは人類の月発見史と同じ時系列で進む**（ガリレオの観測→月面図→ルナ計画→アポロ→月の起源→現代探査→南極の水の発見→火星へ）
- **ゲームをクリアする頃には月の知識が豊富になっている**ことを目指す
- 知識の散りばめ先:博士の解説/街の看板/NPCの会話/ずかんの説明文
- 注意:教育要素は事実の正確さが命。実装前にJAXA・NASA等の信頼できる情報源で裏取りする
- 舞台アイデア:スタート=静かの海（アポロ11号初着陸地）/青白い砂地=南極の氷混じりの砂/終盤=月の裏側/火星ロケットは水資源のある南極から出発

A-1. 主人公が月に来られた理由 → **決定:月面開発プロジェクトに選ばれた**
A-2. 主人公の年齢 → **決定:23歳**
A-3. 博士的キャラ → **決定:キノシタ博士**（やさしくておおらか、どんぶり勘定。最後に売れ残っていたうさもんを譲ってくれる）
A-4. ライバル → **決定:イーゼン**（努力家で地頭がいい、嫌味なエリート役。同じ月面開発プロジェクトの同期という設定が有力）
 - 口癖:**「大変恐縮です」「えぇ」「勝てばダブル、負ければトラブル」**（慇懃無礼な意識高い系）
A-5. 敵組織 → **決定:ヴォイス**（南極の水を独占するために権力を持とうとする組織。水=月の生命線&ロケット燃料であり、独占=月の支配。実際の宇宙開発課題とも重なる）
A-6. 舞台全体の呼び名 → **決定:ムーンワールド**（地名そのものは実在の月面地名を使用）
A-7. ジムの数 → **決定:有名どころの海8つを選んでジム化**（「全ての海の制覇」は完成後の拡張案として保留）
 - 8つの海の候補案（※名称は実装前に要裏取り）:静かの海/晴れの海/雨の海/危難の海/豊かの海/雲の海/湿りの海/神酒の海
A-8. 各ジムの場所・タイプ・リーダー・バッジ名 → 8つの海が確定したら詰める
A-9. 三星王 → **決定:夏の大三角の星座がモチーフ**
 - ベガ（こと座）/アルタイル（わし座）/デネブ（はくちょう座）の3人
 - 三星王=実在する「3つの一等星」と完全一致。七夕（織姫・彦星）とも繋がり、教育コンセプトと好相性
 - キャラ名を星名そのままにするか、もじるかは後で決定
A-10. チャンピオン → **決定:置く**（三星王3連戦→チャンピオンの流れ。誰がチャンピオンかはストーリーと絡めて後で決定）
A-11. 「宇宙一のアルモンマスター」の定義 → **決定:シリーズ全体の壮大な目標**
 - チャンピオンを倒す → 新規プロジェクトへの参画権を得る → 火星、その先の惑星へ
 - 最終目標:**太陽系全体を調査し、人類が住めるインフラを構築して、太陽系全体を人類の住処にする**
 - 月編はその第一章（だから月の知識を学べる構成になっている）
A-12. 火星への伏線 → **決定:NPCの会話で散りばめる**（南極の水=ロケット燃料の噂、火星計画の噂話 など）

## B. アルモン（モンスター）まわり

B-1. 完成版の残り5体 → **決定!**
 - **メテコ**（炎）:隕石から生まれた火の玉。せっかちで熱血。進化:メテコ→メテオドン
 *教育ネタ:月のクレーターは隕石衝突でできた*
 - **シャクリン**（氷）:南極シャクルトンクレーターの永久影に住むペンギン風。恥ずかしがり屋。進化:シャクリン→シャクロス
 *教育ネタ:月の南極に氷がある。実在のシャクルトンクレーターが由来*
 - **ローバウ**（金属）:置き去りにされた月面ローバーがアルモン化した犬型。忠実。進化:ローバウ→ローバーグ
 *教育ネタ:月には歴代の探査機・ローバーが残されている*
 - **ホタルナ**（光）:地球照の光を集めて光る蛍。おっとり優しい。進化:ホタルナ→ルナルクス
 *教育ネタ:「地球照」=三日月の暗部が地球の反射光でうっすら見える実在の現象*
 - **チーズン**（ガス・自由枠）:「月はチーズでできている」の言い伝えから生まれたチーズの精。マイペースな愛されネタ枠。進化は「なし or チーズキング」で後日決定
 *教育ネタ:西洋の古い民話「月はチーズ説」*
B-2. 残り5体の進化 → **決定:チーズンは進化しない**（愛されキャラ枠）。他4体は1段階進化

 - 45億年前に地球と衝突し月を生んだ存在（ジャイアント・インパクト説）
 - **決定:ゲーム内タイプ=金属+砂の複合タイプ**
