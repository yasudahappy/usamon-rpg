# コペルニクス洞窟編（寄り道・光ダンジョン）設計 v1

セレネタウン（晴れの海・光）から入れる寄り道ダンジョン。ジム4（光）への前振りと、
新・光アルモン「レイモグラ」の入手を兼ねる。

凡例：★確定 ／ ☆提案（実装時調整可）／ 🎨アート要（やすださんが描く）

---

## 0. コンセプト
- 月で最も目立つ「光条（レイ）」クレーター **コペルニクス** の地下。
- 若い衝突跡で、放射状に明るい筋（レイ）が伸びる——これを「くらやみに光る星石（ひかりいし）」で表現。
- 暗い洞窟を、点在する **ひかりいし** の灯りをたよりに探索する。

## 1. 教育フック（実在の月ネタ）
- コペルニクスは直径約93km、比較的「若い」（約8億年）大きな衝突クレーター。
- 明るい **光条（ray system）** が放射状に数百km広がり、満月時によく見える。
- 中央丘や段々になった内壁（テラス）を持つ、典型的な大型クレーター。
- ネタは 看板／NPC／ずかん説明文に散りばめる（P2）。

## 2. 場所・接続 ★実装済み(P1)
- 入口：**セレネタウン (7,5)** の北西の岩肌に「ほらあな」。踏むと `copernicus_cave (12,17)` へ。
- 出口：`copernicus_cave (12,19)` → セレネタウン (7,6)。
- マップ：`copernicus_cave.json`（26×20、暗闇＋ひかりいし）。`MAP_KEYS` 登録済み。

## 3. しかけ ☆
- **暗闇**：`placeCaveDarkness()` を流用（プレイヤー中心の視界オーバーレイ）。
- **ひかりいし（タイル123）**：暗闇より上のレイヤーで ぼうっと明滅する道しるべの灯り。
- （P2）**光パズル**：ひかりいしに光を通す/反射させて最奥の扉を開ける（ジム3のセグメント方式を流用予定）。

## 4. 新・光アルモン：レイモグラ（🎨アート要）
- コンセプト：コペルニクスの地下を掘り進む「光のもぐら」。掘った跡が光条のように光る。
- タイプ：**光**。役割：すばやさ寄りのアタッカー（☆）。
- 入手：最奥チャンバー（現状 (13,3) 付近）でレア出現 or もらえる（P2で決定）。
- 🎨必要アート：正面（バトル敵・手持ち・図鑑）＋背面（自分の繰り出し用）。緑背景でOK。
- 実装：`monsters.json` に No.029 で追加（P2、到着まで仮スプライト）。

## 5. やすださんが描くアート

- **レイモグラ**：正面＋背面。
