## 0. 道のり（★確定：溶岩洞を貫通させる）
```
タテアナ村 ──(縦孔ハシゴ)── 溶岩洞（既存・ツキヤマ救出）
                                │ 東奥の岩(救出後に開通)
                                ▼
                     溶岩洞・深部 lava_tube_deep（新）
                                │ 出口ハシゴ
                                ▼
                     リルの谷 rill_route（新・地上ルート）
                                │ 北へ
                                ▼
                ミノリタウン minori_town（新）★ジム3(炎・ヒノ)
```
- 溶岩洞の奥への通路は**ツキヤマ救出後（`pit_rescue_cleared`）に開通**。それまでは岩でふさがっている（調べると「がんばれば通れそうだが、いまは崩れそうだ…」）。
- タテアナ村・溶岩洞がそのまま本編の通り道になり、豊かの海の火山地形（溶岩洞・リル）へテーマが連続する。

## 4. ジム3のしかけ：溶岩バルブ（★スイッチ発展形）
- ジム床は玄武岩。**溶岩チャンネル（tile 100・通行不可）**が横に3本走る。
- **バルブ×2**（調べると回せる）。フラグ `gym3_valve_1` / `gym3_valve_2`（pickups で永続、バトル後も維持）。
  - チャンネル1：バルブ1を回すと冷えて**橋（tile 101・通行可）**になる
  - チャンネル2：バルブ2で冷える
  - チャンネル3（宝ルート）：**バルブが片方だけONのとき**冷える（両方ONだと再加熱）→ カプセル回収は順番を工夫
- 実装は `applyGym3LavaState()`（氷ジム扉と同じ**冪等パターン**：floor/collision/タイル画像を両方向に同期）。
- 看板：「バルブで マグマの ながれを きりかえろ！」
