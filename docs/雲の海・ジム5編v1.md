# 雲の海・ジム5編（くものうみタウン／ガス）設計 v1

凡例：★実装済み ／ ☆提案 ／ 🎨アート

## 1. きりのたに（kiri_valley）★実装済み
- セレネタウン (43,6) → kiri (10,24)。北の出口 (10,0) → くものうみタウン。
- 霧のヘイズ＋ただよう雲の演出。ガス野生（mochichi/mochigori/gorimocchi Lv33-38）。
- トレーナー：たびびと ナギ／ガスけんきゅういん ホノ（仮絵）。
- 教育看板：月の「海」は水ではなく暗い玄武岩の平原。空気がないので雲もできない。

## 2. くものうみタウン（cloud_town）★実装済み（アート仮）
- Mare Nubium のほとりの町。到着カットシーンあり。
- リーダー **クモナ** がジム前に立つ（プレビュー会話）。ジム扉は「かいかん準備中」。
- 教育看板：雲の海の由来。

## 3. リーダー クモナ（🎨アート＝やすださん提供・組込済み）
