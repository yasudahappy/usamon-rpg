# うさもんRPG こだわりアート生成指示書 v1

土台タイルは「Ninja Adventure Asset Pack（CC0）」を使用。
この指示書は、その上に重ねる**月面らしいこだわりパーツ**をAIで生成するためのもの。

## 共通ルール

- **形式**: 見下ろし型2D（top-down / bird's-eye view）のゲーム用タイル・スプライト
- **画風**: GBA世代のドット絵、ルビー・サファイア風。太めの輪郭、はっきりした色、立体感のある陰影
- **解像度**: ドットがくっきりしたピクセルアート。1タイル=32×32ピクセル基準
- **背景**: 透過。タイルの場合はシームレス（seamless / tileable）
- **禁止**: 写実的な3D、ぼやけた絵、実写、既存キャラの流用

## 生成のコツ

1. 1個ずつ生成する
2. 「seamless / tileable」がタイルでは超重要
3. 色が違ったら調整指示して作り直す
4. できた画像は assets/ フォルダに入れる
5. ファイル名は英語でわかりやすく
