# ジム4への道 v1（タウルスさんどう）

ミノリタウン（ジム3）→ 晴れの海タウン（ジム4・光・セレネバッジ）への道のり。
今回は**道のり＋イベント**まで。晴れの海タウンとジム4本体・キヨハラ戦①は次フェーズ。

## 0. 地理と教育フック
- **晴れの海（Serenitatis）**の南の縁には**タウルス山地**が実在し、その谷（タウルス・リットロウ渓谷）に**アポロ17号**が着陸した（1972年・最後の有人月面着陸）。
- 山道の途中の**展望スポット**から晴れの海を一望：「あの谷に アポロ17号が降りた。人類が 最後に 月を歩いた 場所」（ユージン・サーナンの最後の足あと）。
